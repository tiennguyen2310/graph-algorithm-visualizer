Graph Algorithm Visualizer

Welcome!

This tool is designed to help you visualize how different graph algorithms work. Select an algorithm from the dropdown menu above to get started.

How It Works:

  1. Select an Algorithm: Use the dropdown menu at the top-right to choose the algorithm you want to see.
  2. Input a Graph: On each algorithm's page, you will find a text area to define the graph's nodes and edges.
  3. Set Parameters: Some algorithms require extra information, like a source and sink node. You will be prompted for these.
  4. Visualize: Click the "Run" button to see the algorithm animate the process and highlight the solution on the graph.

Available Algorithms:

  • Strongly Connected Components (Kosaraju's Algorithm): Finds and colors the subsets of a directed graph where every vertex is reachable from every other vertex in the same subset.
  • Max-Flow (Edmonds-Karp Algorithm): Determines the maximum possible flow from a source node to a sink node in a weighted, directed graph.
