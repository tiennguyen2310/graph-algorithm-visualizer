Welcome!

This tool is designed to help you visualize how different graph algorithms work. Follow the steps below to get started.

Instructions:

  1. Input Your Graph: In the "Graph Input" text area on the left, define the connections (edges) of your graph.
    • For unweighted graphs, use the format u v (e.g., A B).
    • For weighted graphs, use the format u v w (e.g., A B 7).
    • Each edge should be on a new line.
  2. Choose Graph Type: Check the "Directed Graph" box if you want edges to have a specific direction (A -> B).
  3. Draw the Graph: Click the "Draw Graph" button to see your graph displayed in this area.
  4. Select an Algorithm: Use the dropdown menu at the top to choose an algorithm to run on your graph.
  5. Run and Visualize: Follow the specific instructions that appear for the chosen algorithm and see the visualization unfold!
